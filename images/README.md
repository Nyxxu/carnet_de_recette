# Images des recettes

Ce dossier contient les photos des recettes, embarquées dans le binaire via `embed.FS`.

## Convention de nommage

Le nom du fichier image doit correspondre au champ `image` du YAML de la recette
(typiquement `<slug>.jpg`).

## Format conseillé

- JPEG compressé, ratio 4:3 ou 16:9
- Largeur 1200 px max
- Optimisé pour le web (~100-200 KB par image)
